# 📊 Новые фичи — ИТОГОВАЯ ВЕРСИЯ

## ✅ ЧТО СДЕЛАНО

### 1️⃣ Статистика по источникам 📊

**Модуль:** `shared/source_stats.py`

**Что делает:**
- Считает посты от каждого RSS источника
- Показывает топ-5 источников
- Показывает процент и средний приоритет

**Где используется:**
- ✅ Веб-дашборд (http://localhost:8080)
- Раздел "📡 Источники (7 дней)"

**Пример:**
```
🏛️ Политика
  ria.ru — 92 постов (52.9%)
  tass.ru — 50 постов (28.7%)
  ...
```

---

### 2️⃣ Недельный дайджест 📰

**Что делает:**
- Каждое воскресенье в 20:00
- Отправляет ТОП-10 новостей за неделю
- Сортировка по приоритету
- Без статистики, только новости

**Где:**
- ✅ @I_Politika (воскресенье 20:00)
- ✅ @eco_steroid (воскресенье 20:00)
- ✅ @Film_orbita (воскресенье 20:00)

**Пример:**
```
📰 ДАЙДЖЕСТ ЗА НЕДЕЛЮ
📅 07.03.2026
========================================

🥇 В Орловской области объявили ракетную опасность
   📡 tass.ru | ⭐ 13

🥈 Катарские власти открыли воздушное пространство
   📡 ria.ru | ⭐ 11

🥉 ...
```

---

### 3️⃣ Экспорт статистики в PDF 📤

**Модуль:** `shared/pdf_export.py`

**Что делает:**
- Создаёт PDF отчёты
- Сохраняет в `/root/projectss/reports/`
- Можно скачать вручную

**Где:**
- Папка: `/root/projectss/reports/`
- Формат: `weekly_digest_YYYYMMDD.pdf`

**Использование:**
```python
from shared import export_stats_to_pdf

pdf_path = await export_stats_to_pdf(stats, "/root/projectss/reports/report.pdf")
```
