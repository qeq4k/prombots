### 1️⃣ Статистика по источникам 📊

**Модуль:** `shared/source_stats.py`

**Что делает:**
- Считает количество постов от каждого источника
- Считает средний приоритет новостей
- Показывает топ-10 источников

**Использование:**
```python
from shared import SourceStats

stats = SourceStats(db_path)
report = await stats.get_report(days=7)
text = stats.format_report(report)
```

**Пример отчёта:**
```
📊 Статистика за 7 дн.

📝 Постов: 174
📡 Источников: 4
⭐ Средний приоритет: 7.5

🏆 Топ источников:
🥇 ria.ru
   📝 92 постов (52.9%)
   ⭐ Приоритет: 8.2
🥈 tass.ru
   📝 50 постов (28.7%)
   ⭐ Приоритет: 7.8
...
```

### 2️⃣ Дайджест за неделю (воскресенье 20:00) 📰

**Модуль:** Встроен в `politika.py`, `economika.py`

**Что делает:**
- Каждое воскресенье в 20:00 генерирует дайджест
- Показывает статистику за неделю
- Показывает топ-5 новостей за неделю
- Экспортирует в PDF и отправляет в канал

**Автоматически:**
- ✅ Запускается при старте бота
- ✅ Ждёт следующее воскресенье 20:00
- ✅ Генерирует и отправляет дайджест

**Пример дайджеста:**
```
📊 Статистика за 7 дн.

📝 Постов: 174
📡 Источников: 4
⭐ Средний приоритет: 7.5

🏆 Топ источников:
🥇 ria.ru — 92 постов (52.9%)

========================================
🔥 ТОП-5 новостей за неделю
========================================

1. В Орловской области объявили ракетную опасность
   📡 tass.ru | ⭐ 13

2. Катарские власти открыли воздушное пространство
   📡 ria.ru | ⭐ 11

📊 Полная статистика за неделю (PDF)
```

### 3️⃣ Экспорт статистики в PDF 📤

**Модуль:** `shared/pdf_export.py`

**Что делает:**
- Создаёт красивый PDF отчёт
- Таблицы с источниками
- Графики (опционально)
- Если нет reportlab — создаёт TXT

**Использование:**
```python
from shared import export_stats_to_pdf

pdf_path = await export_stats_to_pdf(report, "report.pdf")
```

**Зависимости:**
```bash
pip3 install reportlab --break-system-packages
```

## 🔧 Настройки

### Изменить время недельного дайджеста:

В `politika.py`:
```python
# Изменить день (0=понедельник, 6=воскресенье)
days_until_sunday = (6 - now.weekday()) % 7

# Изменить время
next_sunday = now.replace(hour=20, minute=0, second=0, microsecond=0)
#(hour=10) — в 10:00
```

### Изменить период отчёта:

```python
report = await stats.get_report(days=30)  # 30 дней вместо 7
```

### Изменить топ новостей:

```python
# В generate_weekly_digest()
LIMIT 5  # → LIMIT 10
```
