### 2. Срочные новости (urgent_news.py)
**Автопостинг критически важных новостей минуя предложку**

Триггеры для автопостинга:
- 🚨 Военные действия (операции, удары, обстрелы)
- 🚨 Критические заявления (президент, экстренные заявления)
- 💰 Обвал рынков, дефолт, девальвация
- 💰 Ключевые решения ЦБ, санкции

**Параметры:**
- Парсинг: раз в 2 минуты
- Минимальный приоритет: 85/100
- Cooldown между постами: 5 минут
- Проверка дубликатов: 24 часа

**Запуск:**
```bash
python3 urgent_news.py
```

### 3. Улучшенная дедупликация
**global_dedup.py** — строгая нормализация текста:
- Убирает цифры, даты, суммы, проценты
- Нормализует имена, страны, города
- Нормализует организации, валюты
- Учитывает категорию (politics/economy/cinema)

**Новые функции:**
- `normalize_text_for_dedup()` — строгая нормализация
- `get_content_dedup_hash()` — улучшенный хеш контента
- `get_duplicate_stats()` — статистика дубликатов

## 📊 Приоритеты новостей

| Приоритет | Описание | Действие |
|-----------|----------|----------|
| 95-100 | Критические (дефолт, война) | Мгновенный автопост |
| 90-94 | Очень важные (заявления президента) | Автопост |
| 85-89 | Важные (санкции, решения ЦБ) | Автопост |
| < 85 | Обычные | Через предложку |

## 🛡️ Защита от дубликатов

**Многоуровневая система:**

1. **Exact match** — точное совпадение URL
2. **Content hash** — хеш заголовка + контента
3. **Global dedup** — проверка между каналами
4. **Fuzzy match** — нечёткое совпадение (85%)

**Нормализация текста:**
```
"Путин заявил о новых санкциях против США на 50%"
→ "NAME заявил о новых санкциях против COUNTRY на PERCENT"
```
