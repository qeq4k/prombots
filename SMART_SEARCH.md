# 🎯 Умный поиск фильмов — Документация

## ✅ Что добавлено

**Умный поиск** — гибридная система рекомендаций фильмов на основе LLM + БД.

**Особенности:**
- LLM (mistral-nemo) анализирует запрос пользователя
- Поиск в БД по параметрам (жанры, теги, годы, ключевые слова)
- LLM формирует красивую выдачу с рекомендациями
- Кэширование запросов (24 часа)
- Стоимость: ~$0.001 на 100 запросов

---

## 🚀 Как использовать

### В боте:

1. Нажми кнопку **"🎯 Умный поиск"**
2. Или отправь команду: `/smart`, `/рекомендации`, `/подборка`
3. Опиши что хочешь посмотреть
4. Получи подборку фильмов

### Примеры запросов:

```
"хочу что-то like Матрицы но посвежее"
"весёлая комедия на вечер с друзьями"
"фантастика про космос как Интерстеллар"
"ужастик чтобы не уснуть ночью"
"ромком чтобы поплакать"
"что посмотреть с девушкой?"
"фильмы с Ди Каприо"
"что-то новое вышло за 2025-2026"
```

## 🔧 Как это работает

### 1. LLM анализ запроса

```python
Запрос: "хочу что-то like Матрицы но посвежее и весёлое"

↓ LLM (mistral-nemo)

{
    "genres": ["фантастика", "боевик"],
    "tags": ["like:Матрица", "киберпанк"],
    "mood": "весёлое",
    "year_from": 2020,
    "year_to": 2026,
    "keywords": ["матрица", "киберпанк", "свежее"]
}
```

### 2. Поиск в БД

```sql
SELECT * FROM movies
WHERE year >= 2020 AND year <= 2026
AND genre IN ('фантастика', 'боевик')
AND (description LIKE '%матрица%' OR description LIKE '%киберпанк%')
ORDER BY rating DESC
LIMIT 15
```

### 3. LLM форматирование

```
Запрос + Найденные фильмы

↓ LLM (mistral-nemo)

🎬 Вот подборка фильмов похожих на Матрицу:

1. Джон Уик (2014) — экшен, киберпанк...
2. Начало (2010) — реальность vs иллюзия...
...
```

## ⚙️ Настройки

### Изменить модель:

В `checkerrr/recommender/__init__.py`:

```python
self.model = "mistralai/mistral-nemo"  # или "openai/gpt-oss-20b"
```

### Изменить лимит кэша:

```python
# Кэш хранится 24 часа (86400 секунд)
if datetime.now(timezone.utc).timestamp() - cached.get("timestamp", 0) < 86400:
```

### Очистить кэш:

```bash
rm /root/projectss/checkerrr/recommender_cache.json
```

Или через команду (если добавишь):
```python
recommender.clear_cache()
```

## 🐛 Troubleshooting

### Умный поиск не работает:

1. Проверь что API ключ есть в `.env`:
   ```bash
   grep ROUTER_API_KEY /root/projectss/.env
   ```

2. Проверь логи бота:
   ```bash
   pm2 logs bot_new --lines 50 | grep -i "умный\|smart\|recommender"
   ```

3. Перезапусти бота:
   ```bash
   pm2 restart bot_new
   ```

### Кэш не сохраняется:

Проверь права на запись:
```bash
ls -la /root/projectss/checkerrr/recommender_cache.json
```

### LLM не отвечает:

Проверь API:
```bash
curl -H "Authorization: Bearer $ROUTER_API_KEY" \
     https://routerai.ru/v1/models
```
