# 📝 Changelog - Checkerrr Bot

## [2.0.0] - 2026-03-04

### ✨ Новые функции

#### 🎲 Случайный фильм
- Кнопка в главном меню "🎲 Случайный фильм"
- Выбор случайного фильма из топ-100 по просмотрам
- Callback "random_movie" для inline-использования

#### 📈 Тренды
- Кнопка в главном меню "📈 Тренды"
- Отображение количества пользователей онлайн ("сейчас смотрят")
- Топ фильмов за день/неделю/месяц с переключением периода
- Сервис `TrendsService` для получения статистики

#### 🏆 Достижения
- Кнопка в главном меню "🏆 Достижения"
- 10 достижений для пользователей:
  - 🔍 Первый поиск
  - 👀 Любопытный (10 поисков)
  - 🎬 Киноман (50 поисков)
  - 🏆 Гуру кино (100 поисков)
  - ⭐ Коллекционер (первый в избранном)
  - 🌟 Ценитель (10 в избранном)
  - 💫 Легенда (50 в избранном)
  - 📅 Недельный (7 дней)
  - 🗓️ Месячный (30 дней)
  - 🎉 Годовой (365 дней)
- Сервис `AchievementService` для управления достижениями
- Уведомления о разблокировке достижений

#### 🔔 Уведомления
- Кнопка в главном меню "🔔 Уведомления"
- Вкл/выкл уведомлений пользователем
- Уведомления о новых фильмах
- Уведомления-напоминания о просмотренных фильмах
- Уведомления о достижениях
- Сервис `NotificationService` для рассылок

#### 👍 Реакции
- Inline-кнопки "👍 Like" и "👎 Dislike" под каждым фильмом
- Подсчёт количества лайков/дизлайков
- Отображение реакции пользователя (галочка ✅)
- Сервис `ReactionService` для управления реакциями
- Top фильмов по рейтингу реакций

#### 🔍 Похожие фильмы
- Inline-кнопка "🔍 Похожие" под фильмом
- Алгоритм подбора на основе:
  - Совпадение жанров (вес 3)
  - Совпадение актёров (вес 2)
  - Совпадение режиссёров (вес 2)
- Сервис `SimilarMoviesService` для поиска похожих фильмов

### 🏗️ Архитектурные изменения

#### Роутеры (handlers/)
- `user_commands.py` - базовые команды (/start, /help, /language)
- `new_features.py` - новые функции (случайный фильм, тренды, достижения, уведомления)
- `reactions.py` - реакции и похожие фильмы

#### Middleware (middlewares/)
- `SubscriptionMiddleware` - проверка подписки через класс
- `VisitsLoggingMiddleware` - логирование посещений
- Поддержка aiogram 3.x BaseMiddleware

#### Сервисы (services/)
- `NotificationService` - уведомления и рассылки
- `AchievementService` - система достижений
- `ReactionService` - лайки/дизлайки
- `TrendsService` - тренды и статистика
- `SimilarMoviesService` - похожие фильмы

#### Константы (constants.py)
- Все магические числа вынесены в отдельный файл
- Константы для пагинации
- Константы для кэширования
- Константы для лимитов
- Константы для достижений

#### Типизация (types.py)
- Data classes для всех основных сущностей:
  - `SubscriptionResult`
  - `SearchResult`
  - `MovieInfo`
  - `UserInfo`
  - `AchievementInfo`
  - `ReactionInfo`
  - `TrendInfo`
  - и другие

### 🗄️ Изменения в базе данных

#### Новые таблицы:
```sql
-- Достижения пользователей
user_achievements (user_id, achievement_type, unlocked_at)

-- Реакции на фильмы
movie_reactions (user_id, movie_id, movie_code, reaction_type, created_at)

-- Лог уведомлений
notifications_log (user_id, movie_id, notification_type, sent_at, is_read)

-- Настройки пользователей
user_settings (user_id, notifications_enabled, reminders_enabled)
```

#### Новые методы в Database:
- `unlock_achievement()` - разблокировать достижение
- `is_achievement_unlocked()` - проверка достижения
- `get_user_achievements()` - получить достижения пользователя
- `add_reaction()` - добавить реакцию
- `remove_reaction()` - удалить реакцию
- `get_user_reaction()` - получить реакцию пользователя
- `get_movie_reactions()` - получить реакции на фильм
- `log_notification()` - залогировать уведомление
- `get_all_users_with_notifications()` - пользователи с уведомлениями
- `set_user_notifications()` - настроить уведомления
- `get_user_notifications_enabled()` - проверка настроек уведомлений
- `get_active_users_count()` - активные пользователи онлайн
- `get_movie_genres()` - жанры фильма
- `get_movie_actors()` - актёры фильма
- `get_movie_directors()` - режиссёры фильма
