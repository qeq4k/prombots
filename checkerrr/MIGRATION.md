# 🔄 Миграция на новую версию бота

## 📋 Что изменилось

### ✨ Новые функции:
1. **🎲 Случайный фильм** - рандомный фильм из топ-100
2. **📈 Тренды** - популярные фильмы + "сейчас смотрят"
3. **🏆 Достижения** - система ачивок за активность
4. **🔔 Уведомления** - о новых фильмах и напоминания
5. **👍 Реакции** - лайки/дизлайки на фильмы
6. **🔍 Похожие фильмы** - рекомендации на основе жанров/актёров/режиссёров

### 🏗️ Архитектурные улучшения:
1. **Роутеры** - код разбит на модули по функциям
2. **Middleware** - проверка подписки и логирование вынесены в отдельные классы
3. **Сервисы** - бизнес-логика отделена от обработчиков
4. **Type hints** - добавлена типизация через dataclasses
5. **Константы** - все магические числа вынесены в constants.py

### 📁 Новая структура:
```
checkerrr/
├── bot.py              # Старый бот (работает)
├── bot_new.py          # Новый бот (рефакторинг)

│
├── handlers/           # Обработчики
│   ├── user_commands.py
│   ├── new_features.py
│   └── reactions.py
│
├── middlewares/        # Middleware
│   └── subscription.py
│
└── services/           # Бизнес-логика
    ├── notifications.py
    ├── achievements.py
    ├── reactions.py
    ├── trends.py
    └── similar_movies.py
```

## 🚀 Установка

### 1. Обновите базу данных
Новые таблицы создадутся автоматически при первом запуске.

**ИЛИ** выполните миграцию вручную:
```bash
python3 -c "from database import Database; db = Database('movies.db'); print('✅ БД обновлена')"
```

### 2. Обновите главное меню
В `keyboards.py` уже добавлены новые кнопки:
- 🎲 Случайный фильм
- 📈 Тренды
- 🏆 Достижения
- 🔔 Уведомления

### 3. Запуск нового бота

**Вариант А: Использовать новый bot_new.py**
```bash
python3 bot_new.py
```

**Вариант Б: Обновить старый bot.py**
Скопируйте роутеры из `handlers/` в старый `bot.py` вручную.

```sql
-- Достижения
CREATE TABLE user_achievements (
    user_id INTEGER,
    achievement_type TEXT,

    PRIMARY KEY (user_id, achievement_type)

CREATE TABLE movie_reactions (
    user_id INTEGER,
    movie_id INTEGER,

    PRIMARY KEY (user_id, movie_id)

-- Уведомления
CREATE TABLE notifications_log (
    user_id INTEGER,
    movie_id INTEGER,
    notification_type TEXT,

-- Настройки пользователей
CREATE TABLE user_settings (
    user_id INTEGER PRIMARY KEY,
    notifications_enabled BOOLEAN DEFAULT 1
);
```

## 🎮 Новые команды меню

### 🎲 Случайный фильм
**Кнопка:** "🎲 Случайный фильм"
**Описание:** Показывает случайный фильм из топ-100 по просмотрам

### 📈 Тренды
**Кнопка:** "📈 Тренды"
**Описание:** Показывает:
- Количество пользователей онлайн ("сейчас смотрят")
- Топ фильмов за день/неделю/месяц

### 🏆 Достижения
**Кнопка:** "🏆 Достижения"
**Описание:** Показывает прогресс пользователя:
- Разблокированные достижения ✅
- Заблокированные достижения 🔒

**Список достижений:**
| Иконка | Название | Условие |
|--------|----------|---------|
| 🔍 | Первый поиск | Первый поиск фильма |
| 👀 | Любопытный | 10 поисков |
| 🎬 | Киноман | 50 поисков |
| 🏆 | Гуру кино | 100 поисков |
| ⭐ | Коллекционер | Первый в избранном |
| 🌟 | Ценитель | 10 в избранном |
| 💫 | Легенда | 50 в избранном |
| 📅 | Недельный | 7 дней с ботом |
| 🗓️ | Месячный | 30 дней с ботом |
| 🎉 | Годовой | 365 дней с ботом |

### 🔔 Уведомления
**Кнопка:** "🔔 Уведомления"
**Описание:** Вкл/выкл уведомления:
- О новых фильмах
- Напоминания о просмотренных
- О достижениях

## 👍 Реакции

Добавлены inline-кнопки к фильмам:
- **👍 Like** - поставить лайк
- **👎 Dislike** - поставить дизлайк

Реакции отображаются счётчиком на кнопке.

## 🔍 Похожие фильмы

К каждому фильму добавлена кнопка **"🔍 Похожие"**.

Алгоритм подбора:
1. Совпадение жанров (вес: 3)
2. Совпадение актёров (вес: 2)
3. Совпадение режиссёров (вес: 2)

Возвращается top-5 по score.

## 🐛 Решение проблем

### Ошибка: "table already exists"
Таблицы уже созданы. Это нормально, просто игнорируйте.

### Не работают новые кнопки
Обновите `keyboards.py` или перезапустите бота.

### Не создаются таблицы
Проверьте права доступа к БД:
```bash
chmod 644 movies.db
```

### Ошибка импорта модулей
Убедитесь что все файлы на месте:
```bash
ls -la handlers/ middlewares/ services/
```
