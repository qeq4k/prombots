# 🎬 КиноБот - Telegram бот для поиска фильмов

Бот для поиска фильмов по коду, названию, актёру или жанру с проверкой подписки на каналы.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка
Скопируйте `.env.example` в `.env` и заполните:
```bash
cp .env.example .env
```

**Обязательные поля:**
- `BOT_TOKEN` - токен от @BotFather
- `ADMIN_IDS` - ваш Telegram ID (можно узнать через @userinfobot)
- `CHANNEL1_NAME`, `CHANNEL1_LINK`, `CHANNEL1_ID` - данные канала

### 3. Заполнение базы фильмов

**Вариант А: Импорт из CSV (ваши данные)**
```bash
# Файл movies_import.csv уже содержит 52 фильма
python3 import_csv.py
```

**Вариант Б: Тестовые фильмы**
```bash
python3 fill_db.py
```

### 4. Запуск бота
```bash
python3 bot.py
```

## 🔧 Конфигурация (.env)

### Основные:
| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен бота |
| `ADMIN_IDS` | ID админов (через запятую) |
| `DATABASE_PATH` | Путь к БД |

### Каналы (до 3):
| Переменная | Описание |
|------------|----------|
| `CHANNEL1_NAME` | Название канала |
| `CHANNEL1_LINK` | Ссылка на канал |
| `CHANNEL1_ID` | ID канала (@name или число) |

### Кэширование:
| Переменная | По умолчанию |
|------------|--------------|
| `USE_REDIS` | false |
| `CACHE_TTL` | 60 |
| `SEARCH_CACHE_TTL` | 600 |

## 📊 База данных

### Таблицы:
- `movies` - фильмы (код, название, ссылка, год, жанры, актёры)
- `genres` - жанры
- `movie_genres` - связь фильмов и жанров
- `movie_actors` - связь фильмов и актёров
- `users` - пользователи
- `search_history` - история поисков
- `favorites` - избранное
- `view_analytics` - просмотры
- `channels` - каналы для подписки

## 🐛 Решение проблем

### Бот не запускается:
1. Проверьте `BOT_TOKEN` в `.env`
2. Установите зависимости: `pip install -r requirements.txt`

### Не работает проверка подписки:
1. Бот должен быть **администратором** в канале
2. Проверьте `CHANNEL_ID` (должен быть @username или числовой ID)

### Пустая база фильмов:
```bash
python3 fill_db.py
```

### Ошибки кэша:
- При отсутствии Redis используется in-memory кэш (автоматически)
