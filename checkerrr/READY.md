- `bot_new.py` - новый бот (с новыми фичами)

**НЕ ЗАБУДЬТЕ остановить старого бота перед запуском нового!**

```bash
# Найти процесс старого бота
ps aux | grep bot.py

# Остановить
kill <PID>

# Или через pm2 если используете
pm2 stop bot
pm2 start bot_new.py
```

## 🔧 Если что-то пошло не так

### Ошибка "table already exists"
Нормально, таблицы уже созданы. Игнорируйте.

### Ошибка "Conflict: terminated by other getUpdates"
Остановите старого бота:
```bash
pkill -f "python3 bot.py"
# или
pm2 stop bot
```

### Ошибка импорта
Убедитесь что все файлы на месте:
```bash
ls -la handlers/ middlewares/ services/
```

### Бот не реагирует на команды
Проверьте токен в `.env` и что бот запущен:
```bash
ps aux | grep bot_new
```
